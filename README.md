# hair-brush-reminder-windows-version
Novimatrem's hair brushing reminder script, for the getting my hair brushed at the right time.

Note: script functionality requires system to remain unmuted and unlocked

# Windows version

Ensure you have installed the required dependency: Git Bash (https://github.com/git-for-windows/git/releases)

Then run hair-brush-reminder-windows-version\windows-version\run-background-win.vbs
